Find the space complexity of:
Alg test(n) {
	if(n > 0) {
		print n;
		test(n - 1);
		test(n - 1);
	}
     }

Solution: 
   test(n-1)
   test(n-1)

test(n) → test(n-1) → test(n-2) → ... → test(1) → test(0)
Number of frames in the deepest chain: n+1 because we include test (0)
Therefore, the maximum number of stack frames is proportional to n.
#frames × O(1) = (n + 1) × O(1) = O(n). [Remove the co efficients to remain with n.]
Maximum simultaneous frames = depth of recursion = n + 1.

Space Complexity = O(n).
